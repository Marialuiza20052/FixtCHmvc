# 🔧 FixTech MVC

Sistema de gerenciamento para uma assistência técnica desenvolvido em ASP.NET Core MVC.

## 📖 Sobre

O FixTech MVC foi desenvolvido como projeto final da disciplina utilizando o padrão MVC. O sistema permite o gerenciamento de serviços de uma assistência técnica.

## 🚀 Funcionalidades

- Página Inicial
- Página Sobre
- Página Contato
- Login Administrativo
- Cadastro de Serviços
- Listagem de Serviços
- Pesquisa de Serviços
- Editar Serviços
- Excluir Serviços
- Visualizar Detalhes

## 🛠️ Tecnologias Utilizadas

- ASP.NET Core MVC
- C#
- Entity Framework Core
- SQLite
- Bootstrap
- HTML5
- CSS3

## 📂 Estrutura do Projeto

```
Controllers/
Models/
Views/
Data/
Migrations/
wwwroot/
```

## ▶️ Como Executar

1. Clone este repositório.
2. Abra o projeto no Visual Studio 2022.
3. Execute a aplicação.
